本地调试 npm 模块

前言

在项目开发过程中，项目会依赖一些独立发布的模块，模块的变更，需要结合项目进行联调，我们不可能发布 npm 上，再进行联调，原则上，需要在本地进行模块验证，通过后方可将模块发布到 npm 上。

本地如何调试 npm 模块

npm link 用来在本地项目和本地 npm 模块之间建立连接，可以在本地进行模块联调。

在模块目录下，执行 npm link ，会在 npm 的全局安装目录下，创建软链接指向当前模块的目录。

npm 全局安装目录一般为 {prefix}/lib/node_modules，其中的 prefix 可以用 npm prefix -g 来查看。

在项目目录下，执行 npm link my-module-name ，会在当前项目的 node_modules 下创建一个软链接，指向全局安装的 my-module 。
